# @fy-/icons
Social &amp; custom SVG icons for vue (optimized one path svg icons)
```
  pnpm add @fy-/icons
```

## Usage
```javascript
import { GoogleIcon } from "@fy-/icons";
```
```vue
<GoogleIcon class="w-5 h-5 text-blue-600" />
```
